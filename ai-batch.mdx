## AIバッチ処理の基本

AIバッチ処理は、複数の行に対してAI処理を一括で実行できる機能です。この機能を使用することで、大量のデータを効率的に処理することができます。以下では、基本的な使用方法について説明します。

## 処理の実行手順

### 1. 処理対象の選択
処理を開始する前に、まず処理対象となる行を選択する必要があります。check列にあるチェックボックスを使用して、処理したい行を指定します。

複数の行を効率的に選択するために、以下のショートカットが用意されています：
- `Shift + クリック`を使用すると、クリックした範囲の行を一括で選択できます
- `Command + クリック`で、離れた位置にある行を個別に選択することも可能です

### 2. 処理の実行と進捗確認
処理対象の選択が完了したら、画面上部にある「AIバッチ処理実行」ボタンをクリックして処理を開始します。処理が開始されると、進捗状況がリアルタイムで表示されるため、処理の状態を常に確認することができます。

また、処理の実行中に問題が発生した場合や、設定を見直したい場合は、「キャンセル」ボタンをクリックすることで、いつでも処理を中断することができます。

### 3. 処理結果の確認
処理が完了すると、結果は自動的にoutput列に反映されます。処理中にエラーが発生した場合は、エラーメッセージが表示され、どの行でどのような問題が発生したのかを確認することができます。

## 便利なキーボードショートカット

効率的な操作をサポートするため、以下のキーボードショートカットを用意しています：

| 操作 | ショートカット | 説明 |
|------|----------------|------|
| 処理実行 | Command + Shift + Enter | 選択した行のAI処理を開始します |
| 変更承認 | Command + Enter | 処理結果を承認します |
| キャンセル/変更却下 | Command + Backspace | 処理をキャンセルまたは結果を却下します |

<Tip>
初めてAIバッチ処理を使用する場合は、大量のデータを一度に処理する前に、まず少数の行（5-10行程度）で動作確認を行うことをお勧めします。これにより、設定が意図した通りに機能しているかを安全に確認することができます。
</Tip>
