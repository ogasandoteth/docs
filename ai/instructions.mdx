## システム指示の基本

### 役割の定義
```text
あなたは[役割]です
例: あなたは優秀なキャリアコンサルタントです
```

### 制約の設定
```text
回答は[制約]でお願いします
例: 回答は必ず敬語で書いてください
```

### 出力形式の指定
```text
回答は以下の形式で出力してください：
[形式の詳細]
```

## プロンプトの書き方

### 基本構造
1. 目的の明確化
2. 具体的な指示
3. 期待する出力形式

### 変数の使用
入力データを参照する場合は `{変数名}` を使用：
```text
{企業名}様への提案書を作成してください
```

<Tip>
システム指示とプロンプトを組み合わせることで、より精度の高い出力が得られます。
</Tip>
