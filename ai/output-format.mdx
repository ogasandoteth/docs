# 出力フォーマット

## 基本フォーマット

AIの出力は以下の形式でJSONとして返されます：

```json
{
  "タイトル": "生成されたタイトル",
  "本文": "生成された本文",
  "追加情報": "その他の情報"
}
```

## カスタマイズ

### 出力キーの設定
1. output列のラベルを設定
2. システム指示で出力形式を指定
3. 必要に応じてプロンプトで詳細を指定

### 複数列への出力
同じAI処理結果を複数の出力列に振り分けることができます：

1. 複数のoutput列を作成
2. 各列に異なるラベルを設定
3. システム指示でJSONキーを対応させる

<Warning>
出力フォーマットが不正な場合、エラーとなる可能性があります。
</Warning>
