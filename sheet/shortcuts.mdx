## 編集操作

| 操作 | Windows | Mac |
|------|---------|-----|
| セル編集開始 | Enter | Return |
| 編集確定 | Ctrl + Enter | ⌘ + Return |
| 編集キャンセル | Esc | Esc |
| 次のセルへ | Tab | Tab |
| 前のセルへ | Shift + Tab | Shift + Tab |

## AI処理

| 操作 | Windows | Mac |
|------|---------|-----|
| バッチ処理実行 | Ctrl + Shift + Enter | ⌘ + Shift + Return |
| 処理キャンセル | Ctrl + Backspace | ⌘ + Backspace |

<Note>
これらのショートカットは設定画面からカスタマイズ可能です。
</Note>
