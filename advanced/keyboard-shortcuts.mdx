効率的な操作のために、以下のショートカットキーを用意しています。

## 基本操作

| 操作 | ショートカット | 説明 |
|------|----------------|------|
| 処理実行 | `Command + Shift + Enter` | 選択した行のAI処理を実行します |
| 変更承認 | `Command + Enter` | 編集内容を確定します |
| キャンセル/変更却下 | `Command + Backspace` | 現在の操作をキャンセルします |

## シート操作のヒント

### 効率的な選択操作
- `Shift + クリック`: 範囲選択
- `Command + クリック`: 複数選択
- `Command + A`: 全選択

### 編集操作
- `Command + C`: コピー
- `Command + V`: 貼り付け
- `Command + X`: 切り取り
- `Command + Z`: 元に戻す

<Note>
Windows PCをご利用の場合は、`Command`キーの代わりに`Ctrl`キーをご使用ください。
</Note>
